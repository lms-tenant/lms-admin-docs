# Bienvenido al Panel de Administración

Esta documentación te guía para configurar y gestionar tu academia online desde cero. No necesitás conocimientos técnicos — todo se hace desde el panel de administración.

## ¿Por dónde empezar?

Si es tu primera vez, seguí este orden:

1. [Configurá tu academia](primeros-pasos/configuracion-inicial.md) — nombre, logo y colores
2. [Creá tu primer curso](cursos/crear-curso.md) — estructura, módulos y lecciones
3. [Invitá a tus estudiantes](usuarios/invitar-estudiantes.md) — manualmente o por link
4. [Configurá la tienda](tienda/configurar-tienda.md) — precios y métodos de pago
