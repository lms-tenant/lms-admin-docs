# Ver ventas y órdenes

## Listado de órdenes

Desde **Tienda → Órdenes** podés ver todas las compras realizadas con:

- Fecha y hora
- Nombre del estudiante
- Producto comprado
- Monto pagado
- Estado: Completada / Pendiente / Cancelada

## Filtros útiles

- Por rango de fechas
- Por producto
- Por estado de la orden

## Detalle de una orden

Hacé clic en cualquier orden para ver:

- Datos del comprador
- Método de pago usado
- ID de transacción (útil para consultas con MercadoPago o PayPal)
- Productos incluidos

## Métricas de ventas

Para un resumen visual de ingresos, usá el **Dashboard** o la sección **Analíticas → Métricas de ventas**.

→ Ver [Métricas de ventas](../analiticas/metricas-ventas.md)
