# Cupones y descuentos

Los cupones permiten ofrecer descuentos a segmentos específicos de estudiantes.

## Crear un cupón

Desde **Tienda → Cupones → Nuevo cupón**:

1. **Código** — el código que el estudiante ingresa al pagar (ej: `BIENVENIDO20`)
2. **Tipo de descuento**:
   - **Porcentaje** — ej: 20% de descuento
   - **Monto fijo** — ej: $500 de descuento
3. **Aplica a** — todos los cursos, cursos específicos o planes de suscripción
4. **Límite de usos** — cuántas veces puede usarse en total (dejar vacío para ilimitado)
5. **Un uso por estudiante** — cada estudiante solo puede usar el cupón una vez
6. **Vencimiento** — fecha hasta la cual es válido

## Casos de uso comunes

- **Lanzamiento** — `LANZAMIENTO30` — 30% off para los primeros compradores
- **Referidos** — código personalizado por cada persona que refiere estudiantes
- **Alumni** — descuento para ex-estudiantes que compran un nuevo curso

## Ver uso de cupones

Desde **Tienda → Cupones** podés ver cuántas veces se usó cada cupón y el ahorro total generado.
