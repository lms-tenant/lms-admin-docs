# Módulos y lecciones

La estructura de un curso es: **Curso → Módulos → Lecciones**.

## Módulos

Los módulos agrupan lecciones relacionadas (equivalen a capítulos o semanas).

**Crear un módulo:**
1. Dentro del editor del curso, hacé clic en **+ Agregar módulo**
2. Ingresá el nombre del módulo
3. Guardá

**Reordenar módulos:** arrastrá el ícono ≡ a la posición deseada.

## Lecciones

Dentro de cada módulo podés agregar lecciones. Cada lección es una unidad de contenido independiente.

**Crear una lección:**
1. Dentro del módulo, hacé clic en **+ Agregar lección**
2. Elegí el tipo de lección (ver abajo)
3. Completá el contenido
4. Guardá

**Reordenar lecciones:** igual que los módulos, arrastrá el ícono ≡.

## Tipos de lección

| Tipo | Descripción |
|---|---|
| Video | Embed de YouTube, Vimeo u otros |
| Texto | Editor rich text con imágenes y código |
| Examen | Preguntas con corrección automática |
| Proyecto | Entrega de archivos o links por parte del estudiante |

→ Ver [Tipos de lecciones](tipos-de-lecciones.md) para la configuración detallada de cada uno.

## Visibilidad de lecciones

- **Pública** — visible para todos (útil para lecciones de muestra)
- **Privada** — solo para estudiantes inscriptos
- **Vista previa** — visible sin estar inscripto, pero sin poder interactuar
