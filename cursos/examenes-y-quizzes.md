# Exámenes y quizzes

Los exámenes permiten evaluar a los estudiantes con corrección automática. Pueden ser obligatorios para obtener el certificado.

## Crear un examen

1. Dentro de un módulo, hacé clic en **+ Agregar lección**
2. Seleccioná el tipo **Examen**
3. Ingresá el nombre del examen
4. Hacé clic en **Editar preguntas**

## Tipos de preguntas

| Tipo | Descripción |
|---|---|
| Opción múltiple (una respuesta) | El estudiante elige una sola opción correcta |
| Opción múltiple (varias respuestas) | Puede haber más de una respuesta correcta |
| Verdadero / Falso | Dos opciones posibles |
| Texto libre | El estudiante escribe; sin corrección automática |

## Configurar una pregunta

1. Hacé clic en **+ Agregar pregunta**
2. Escribí el enunciado
3. Agregá las opciones de respuesta
4. Marcá la(s) respuesta(s) correcta(s)
5. (Opcional) Agregá una explicación que se muestra después de responder

## Configuración del examen

- **Puntaje mínimo para aprobar** — porcentaje requerido (ej: 70%)
- **Intentos permitidos** — cuántas veces puede rendir el estudiante
- **Tiempo límite** — en minutos (dejar vacío para sin límite)
- **Mostrar respuestas al finalizar** — el estudiante ve qué contestó bien/mal

## Exámenes y certificados

Si el curso tiene certificado habilitado, podés requerir que el estudiante apruebe todos los exámenes marcados como obligatorios antes de emitirlo.
