# Métodos de pago

La plataforma soporta MercadoPago y PayPal. Podés tener ambos activos simultáneamente.

## MercadoPago

Recomendado para academias en Argentina, México, Brasil y otros países de Latinoamérica.

**Para configurarlo:**
1. Ingresá a [mercadopago.com](https://mercadopago.com) y creá una cuenta de vendedor
2. Desde el panel de MercadoPago, generá tus **credenciales de producción** (Public Key y Access Token)
3. En tu academia: **Configuración → Integraciones → MercadoPago**
4. Pegá el Public Key y el Access Token
5. Guardá y probá con una compra de prueba

> **Importante:** Usá las credenciales de **producción**, no las de prueba (sandbox), para recibir pagos reales.

## PayPal

Recomendado para ventas internacionales.

**Para configurarlo:**
1. Ingresá a [developer.paypal.com](https://developer.paypal.com) y creá una app
2. Obtené el **Client ID** y el **Secret**
3. En tu academia: **Configuración → Integraciones → PayPal**
4. Ingresá las credenciales
5. Guardá

## Comisiones

Las comisiones las cobra directamente la pasarela de pago (MercadoPago o PayPal). La plataforma no cobra comisión adicional sobre las ventas.
