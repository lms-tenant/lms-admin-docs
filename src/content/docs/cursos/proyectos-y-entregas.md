# Proyectos y entregas

Los proyectos permiten asignar trabajo práctico y recibir entregas de los estudiantes.

## Crear un proyecto

1. Dentro de un módulo, agregá una lección de tipo **Proyecto**
2. Escribí las instrucciones en el editor rich text
3. Configurá el tipo de entrega:
   - **Link** — el estudiante pega una URL (repositorio, Drive, Figma, etc.)
   - **Archivo** — el estudiante sube un documento directamente

## Ver entregas

Desde **Usuarios → [nombre del estudiante] → Proyectos** o desde la lección directamente en el editor de cursos.

Cada entrega muestra:
- Fecha y hora de entrega
- Link o archivo enviado
- Estado: Pendiente / Revisado

## Marcar como revisado

1. Abrí la entrega
2. Dejá un comentario (opcional)
3. Hacé clic en **Marcar como revisado**

El estudiante recibe una notificación cuando su entrega es revisada.

## Proyectos y certificados

Podés requerir que el estudiante entregue todos los proyectos del curso antes de emitir el certificado. Se configura desde los ajustes del certificado del curso.
