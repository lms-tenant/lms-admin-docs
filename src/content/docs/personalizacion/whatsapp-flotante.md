# WhatsApp flotante

El botón flotante de WhatsApp aparece en todas las páginas de la academia y permite a los visitantes contactarte directamente.

## Configurar el botón

Desde **Personalización → WhatsApp**:

1. **Número de teléfono** — ingresá el número con código de país (ej: `5491112345678` para Argentina)
2. **Mensaje predeterminado** — texto que se pre-carga cuando el visitante hace clic (ej: *"Hola, quiero saber más sobre los cursos"*)
3. **Páginas donde aparece** — podés mostrarlo en todas las páginas o solo en páginas específicas
4. Guardá

## Número con código de país

El número debe incluir el código de país sin el signo `+`:

| País | Formato |
|---|---|
| Argentina | `549` + código de área + número |
| México | `52` + número |
| España | `34` + número |

Ejemplo: `5491198765432` (Argentina, código de área 11)

## Desactivar el botón

Para ocultar el botón sin eliminar la configuración, desactivá el toggle **Mostrar WhatsApp** y guardá.
