# Branding: logo y colores

El branding define cómo se ve tu academia para los estudiantes. Se aplica en toda la plataforma automáticamente.

## Logo

Desde **Personalización → Branding**:

1. Hacé clic en **Subir logo**
2. Formatos aceptados: PNG, SVG, JPG
3. Tamaño recomendado: **200×60 px** con fondo transparente (PNG o SVG)
4. Guardá — el logo se actualiza inmediatamente en el encabezado

**Logo para modo oscuro:** Podés subir una versión alternativa del logo para el modo oscuro (por ejemplo, en blanco si el logo original es oscuro).

## Colores

- **Color primario** — se usa en botones principales, links activos y elementos destacados
- **Color secundario** — se usa en elementos de acento y hover

Para definir colores:
1. Usá el selector de color visual, o
2. Ingresá el código hexadecimal directamente (ej: `#10b981`)

Guardá los cambios y recargá la página para ver el resultado.

## Favicon

El favicon es el ícono que aparece en la pestaña del navegador. Subilo desde **Personalización → Branding → Favicon** en formato ICO o PNG de 32×32 px.

## Vista previa

Los cambios de branding se aplican en tiempo real. Abrí la academia en otra pestaña para ver cómo quedan sin cerrar el panel.
