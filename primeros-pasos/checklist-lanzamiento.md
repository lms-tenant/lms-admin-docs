# Checklist de lanzamiento

Usá esta lista antes de abrir tu academia a los estudiantes.

## Configuración básica

- [ ] Nombre de la academia configurado
- [ ] Logo subido
- [ ] Colores de marca definidos
- [ ] Zona horaria correcta

## Contenido

- [ ] Al menos un curso publicado
- [ ] Cada curso tiene al menos una lección
- [ ] Las lecciones tienen video o contenido de texto
- [ ] Portadas de cursos subidas

## Tienda y pagos

- [ ] Método de pago configurado (MercadoPago o PayPal)
- [ ] Al menos un producto o plan de precios creado
- [ ] Probaste una compra de prueba

## Usuarios

- [ ] Invitaste a un estudiante de prueba
- [ ] El estudiante puede inscribirse y ver lecciones

## Comunicación

- [ ] Correos automáticos funcionando (verificá en la bandeja de entrada)
- [ ] WhatsApp flotante configurado (si aplica)

---

✅ Si tildaste todo, ¡tu academia está lista para recibir estudiantes!
